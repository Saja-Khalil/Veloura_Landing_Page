# Veloura

## About the Project

Veloura is a simple landing page that I designed and developed using HTML and CSS.

The idea of the store is to combine elegant mugs with authentic Korean skincare products in one place. My goal was to create a clean, soft, and user-friendly design with warm colors and a simple layout.

## Features

* Responsive navigation bar
* Hero section with a welcome message and image
* About Us section
* Services section with three service cards
* Contact form
* Footer
* Hover effects on buttons and cards
* Soft color palette for a modern look

## Technologies Used

* HTML
* CSS
